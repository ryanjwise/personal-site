What is CSS?

CSS Documents allow visual manipulation of HTML content. It adds 'style' to a website and can be used to define and control colour, size, and position of elements.

CSS Stands for 'Cascading Style Sheet' and functions by calling out tags, ID's and classes defined within the HTML document, and assigning style sheets to them in order of specificity.

p { Property1: value; Property2: value; Property3: value; } #id { Property1: value; Property2: value; Property3: value; } .class { Property1: value; Property2: value; Property3: value; }
